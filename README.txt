

My Blog
A simple web blog built using HTML, CSS, and Python.

Project Homepage

Introduction
My Blog is a personal project that showcases my skills in web development. It's a simple blog that allows users to read and share articles on various topics.

Features
Responsive design for a great user experience on all devices
Clean and minimalistic design for easy reading
Articles are categorized for easy navigation
Users can share articles on social media platforms
Installation
To install the project, follow these steps:

Clone the repository using git clone https://github.com/PavicicJadranko/My-First-Website.git
Open the project in your favorite code editor
Run the project by opening the index.html file in a web browser
Usage
To use the project, simply navigate to the index.html file and open it in a web browser. You can then browse through the articles and share them on social media platforms.

Contributing
